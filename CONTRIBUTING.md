# Contributing to Awesome AI-Native Jobs

Thanks for helping map the AI-native job market! Contributions keep this list useful.

## What to add

- **New roles** that fit the "AI-native" theme (with what they do + a source).
- **Companies hiring** for the listed roles.
- **Interview questions** you were actually asked (anonymized).
- **Projects / resources** that genuinely help people land these roles.
- **Corrections** to salary ranges, skills, or links.

## How to contribute

1. Fork this repo.
2. Edit the relevant section (keep the existing format + alphabetical/logical order).
3. Keep entries **specific and genuinely useful** — no thin promo or low-quality links.
4. Open a Pull Request with a one-line description of what you added.

Prefer not to edit Markdown? Open an **issue** using the "Add a role or resource" template and we'll add it.

## Style

- One idea per line; lead with the concrete value.
- Cite a source for salary/stat claims where possible.
- Keep links live; we periodically prune dead ones.

## Code of conduct

Be kind and helpful. This community exists to get people hired. 💜
